# beautipyQt

Customize PyQt Widgets for your taste!!

## Introduction

PyQt의 위젯들에 디자인을 입혀서 모아두려고 합니다.

마음껏 가져가셔서 사용하셔도 됩니다.

PyQt로도 멋지고 예쁜 소프트웨어를 만들 수 있습니다.
